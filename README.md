# Pawnder Web

Sitio web oficial de Pawnder — La app social para mascotas.

## Páginas

- `/` — Landing page
- `/privacy` — Política de Privacidad
- `/soporte` — Soporte y FAQ

## Stack

- HTML estático
- Tailwind CSS (via CDN)
- Deploy: Vercel
- Dominio: pawnder.pet

## Editar

Cualquier cambio se hace directo en los `.html` y al hacer push a `main`, Vercel redeploy automáticamente.
